All notable changes to this project will be documented in this file.

### Added
- Groq provider
- NVIDIA provider
- `--loop` argument
### Changed
- Test system modified
- `README.md` updated
## [0.3] - 2025-12-17
### Added
- OpenRouter provider
- Cerebras provider
### Changed
- Test system modified
- `README.md` updated
- AI Studio main model changed to `gemini-2.5-flash`
- AI Studio fallback model changed to `gemma-3-1b-it`
- Providers moved to `providers.py`
## [0.2] - 2025-12-05
### Added
- Logo
- `summarize` mode
- `simplify` mode
- `bulletize` mode
- `shorten` mode
### Changed
- `README.md` updated
- Cloudflare fallback model changed to `meta/llama-3.1-8b-instruct-fast`
- Model switching modified
## [0.1] - 2025-11-26
### Added
- `run_mytext` function
- AI Studio provider
- Cloudflare provider
- `--mode` argument
- `--tone` argument
